# Bible Source

- **Translation:** Berean Standard Bible
- **Abbreviation:** BSB
- **Source:** Official Berean Bible download
- **Download page:** https://berean.bible/downloads.htm
- **Imported source:** https://bereanbible.com/bsb.txt
- **Public-domain status:** The source states that the text of God’s Word has been dedicated to the public domain.
- **Import format used:** Official tab-delimited plain-text edition (`Book chapter:verse<TAB>text`)
- **Date imported:** 2026-07-13
- **Conversion performed:** The import script parsed each official verse row and grouped the unchanged text into 66 canonical book JSON files with chapters and verses. It also generated a lightweight manifest.
- **Wording confirmation:** No Bible wording was generated, reconstructed, paraphrased, normalized, or edited. Verse text is written to JSON exactly as it appears after the source row’s tab separator.

## Attribution

The Holy Bible, Berean Standard Bible, BSB is produced in cooperation with Bible Hub, Discovery Bible, OpenBible.com, and the Berean Bible Translation Committee. This text of God's Word has been dedicated to the public domain.

The official download page also offers USFM, USJ, and USX archives. This implementation uses the official text edition because it is directly parseable and preserves the same BSB wording without relying on a third-party dataset.
